## プロジェクト概要
- 対象：みつくる／**ミツクル**（WEBマーケティング ワンストップ支援）のトップページ。
- 成果物（**現行の本実装＝公式ハンドオフバンドル「ミツクル.dc.html」の再現**）：
  - `index.html` ＋ `styles.css` ＋ `script.js` ＋ `assets/`（セマンティックHTML/CSS、モバイルファースト、BP 768px/1024px、Noto Sans JP 400-900）。
  - 参考：`mitukuru-top.html`（初期プロトタイプ、みつくる・濃緑）。`*.dc.html`/`support.js` は元バンドル（dc-runtime用、編集対象外）。
- デザインシステム：`.claude/skills/mitsukuru-design/SKILL.md`（ミツクルのライトグリーン `#7CB342` トークン・コンポーネント・レスポンシブ規約。バンドルから抽出）。

### 本実装（index.html / ミツクル）の構成・仕様
- ブランド：**ミツクル**。配色は**ライトグリーン `#7CB342`**（濃 `#689A38/#5F9233`、淡 `#FAFBF7/#EFF6E4/#F6FAEF`、文字 `#2B2B2B`/`#6B7280`）。見出しは**900**。
- セクション順：ヘッダー / ヒーロー / お悩み(＋価値バナー) / 作れるもの(6) / 選ばれる理由(01-03) / 使い方の流れ(4ステップ＋矢印) / 事例(Before→After 3) / パートナー(アバター8) / よくある質問 / 末尾CTA / フッター。
- ヘッダーナビはアンカーリンク（#make/#why/#how/#works/#faq/#cta）。<768px はハンバーガー（`script.js`）。
- FAQは**JSアコーディオン**（`<button aria-expanded>`＋`grid-template-rows:0fr→1fr`、+/−トグル、1つずつ開く）。
- 画像は `assets/`（全29点：`logo.png`/`hero-illustration-trim.png`/`pain-1〜3`/`make-*`/`why-1〜3`/`step-01〜04`/`case-1〜3`/`partner-*`）に配置し相対パス参照。Google ドライブ経由で取得。
- アイコンはインラインSVGスプライト（`#i-*`）＋`<use>`。`:focus-visible`・`prefers-reduced-motion` 対応。


## サービス内容（正確な前提・誇張しない）
3つは**それぞれ独立したサービス**：
1. **AI提案**：ヒアリングシートに答えるだけで、AI生成の提案書（構成提案）＋簡易見積もりが届く。「たたき台」が無料で手に入る。シートが入口。
2. **パートナー協業**：シート記入後に会員登録すると、内容に合うスキルの外部パートナーを紹介。**ディレクション・管理は不要**、1ページ・1投稿から小ロット依頼可。人手不足でも必要なスキルに繋がれる。
3. **動画学習**：内製したいスキルは活用動画で学べる（①②とは独立）。

## デザイン方針（確定事項）
- トーン：**薄いグリーン基調×やわらか**（基調色＝ライトグリーン、ボタン・アクセント＝濃いグリーン `#1F8A4D`、丸み・余白多め、親しみ・手軽さ）。カラートークンは旧 `--coral*` 名のままグリーン値に置換（`--coral`=濃グリーン、`--coral-50/100/200`=ライトグリーン系、`--bg-soft`=#EFF6F0）。
- コピーは**丁寧・親切**な「〜できます／〜いただけます」調。言いきりのAIっぽい見出しは避ける。
- **タイポグラフィ**：見出し・本文とも `Noto Sans JP`（`--font-display`＝`--font-body`＝Noto）。丸ゴシックは不採用。
- パートナーカード／事例サムネのプレースホルダー・グラデーションも緑系（グリーン・ティール・オリーブ等）で統一。
- **シグネチャ要素**：①各セクション見出し（`.h2`/`.final h2`）下の手描き風・波線アンダーライン（data-URI SVG）。②ブランドの核「届く」をマーカーハイライト（`.mark`）で強調。
- **モーション**：スクロールで各 `section.sec` を淡くフェードアップ（JSで `.reveal` 付与）、カードはホバーで浮き上がり。いずれも `prefers-reduced-motion` を尊重。フォーカスは `:focus-visible` で可視化。
- ロゴ：元サイトの `https://mitukuru.com/img/logo_miturukuru.svg` を使用。
- メインビジュアル：ヒーロー右側に背景イラスト（`img/hero-illust.png`、`mix-blend-mode:multiply` で背景になじませる）。イラストは元サイト作風（濃紺#2e384c＋ミント＋肌色＋ベージュ＋ピンク差し色）が基準。

## ページ構成（上→下）

2. ヒーロー：見出し「ヒアリングシートに答えるだけ。／お見積もりと構成提案が、**届く！**」＋CTA2つ。右に背景イラスト。見出し・ボタンは折り返さない（`.l`＋`white-space:nowrap`、640px以下は解除）

4. こんなお悩み、ありませんか？
5. ご提案の後は、最適なパートナーに頼むだけ（`#features`：パートナーカードを横フリック＝横スクロールで多数表示。各カードは2列構成＝左にイラスト枠 `.photo`（現状グラデーション＋人物シルエット。`<div class="photo"><img ...></div>` に差し替え可、`object-fit:cover`）、右にクリエイター名・対応カテゴリ・スキルアイコン（`.skill`＋`<use href="#sk-*">` スプライト）。※既存サイトのクリエイターイラストはネットワークポリシーで取得不可のため未取得）
6. ご相談から生まれた事例（`#works`：事例紹介。`.thumb` がサムネイル画像枠＝現状グラデーション。`auto-fit`グリッド。ナビ「事例」は `#works` を指す）
7. ご利用は、とてもかんたんです（4ステップ）
8. みつくるが、大切にしていること（4点）
9. ご利用者さまの声（3件）
10. 外注だけでなく、社内にもスキルを（動画学習・下部に配置）
11. よくいただくご質問（FAQ、details開閉）

- ページ：`about` / `cases` / `partners` / `partner-detail` / `partner-join` / `videos` / `hearing` / `template` / `mypage` / `login` / `request-detail` / `request-history`。
- 共通サイトナビ（gnav ヘッダー／フッター）を全13ページで統一。内部リンク(href)は英語ファイル名に統一（`ログイン→login` `無料で始める→hearing` `よくある(ご)質問→index.html#faq` `事例→cases` `パートナー紹介→partners` `動画→videos` `詳細を見る→request-detail` 等）。リンク切れ無し（検証済み）。
